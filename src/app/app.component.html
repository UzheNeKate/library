Описание приложения

В приложении работает навигация. При открытии проекта отображается компонент BookCenter, для которого дочерним подгружается компонент BookList. При нажатии на один из элементов списка BookList подгружается соответствующий компонент BookDetails.
Так же реализована реактивная форма с валидации на обязательное заполнение и минимальной и максимальной длинной для поля name
Источником правды выступает сервис books.service, за счет него осушествляется взаимодействие между компонентами.
